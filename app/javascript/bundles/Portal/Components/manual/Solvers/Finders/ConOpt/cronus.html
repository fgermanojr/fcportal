Solver CRONUS

CRONUS applies the method of Sequential Quadratic Programming (SQP):

  • Powell, M. J. D.: "An Efficient Method for Finding the Minimum of a Function of Several Variables without Calculating Derivatives." Computer J., 7 (4), 1964, pp. 303-307;
  • Powell, M. J. D.: "A Fast Algorithm for Nonlinearly Constrained Optimization Calculations." Report DAMTP77/NA2, University of Cambridge, England, 1977.

It provides optional selection of two optimizers and four vector search algorithms. The optimizer options are the same as for ODIN, and the vector search options are the same as for APOLLO.

The general form of the FIND statement for CRONUS is:
FIND unknowns IN modelcall BY CRONUS {(controller)}
  {{WITH|AND} LOWER floor} {{WITH|AND} UPPER ceiling}
  {REPORTING auxiliaries} {{WITH} FLAGsignal}
  {HOLDING inequalities} {MATCHING equalities}
  TO MINIMIZE|MAXIMIZE objective

The modelcall symbol is the model name optionally followed by an argument list as in the right-hand-side of a CALL statement.

Value
The control variables of CRONUS are as follows:
Variable Preset Value Option
OPTIMIZR 1 1 Method of Feasible Directions (MFD)
2 Modified Method of Feasible Directions (MMFD)
VSEARCH 1 1 Golden-section search method
2 Golden-section search+polynomial interpolation
3 Bounded polynomial interpolation
4 Unbounded polynomial interpolation
ABORT 1 0 Delay a model abort until the current iteration is complete.
1 Execute a model abort after the current step in which an ABORT statement was executed.
DETAIL 1 0 No detailed iteration point
n Detailed print every nth iteration plus first and last
SUMMARY 1 1 Print iteration summary
0 No iteration summary
BREAKIN 0 0 No interactive breakpoints
n Breakpoint after each nth iteration
DETOUT +1 0 Detailed report to PRINTER
+1 Detailed report to SCROLL
-1 Detailed report to Console
SUMOUT +1 0 Summary report to PRINTER
+1 Summary report to SCROLL
-1 Summary report to Console
REDO niv+1   Restart parameter for conjugate direction and variable metric methods. Unconstrained minimization is restarted with a steepest descent direction every REDO iterations.
SCALE 1 0 No scaling is done.
1 Design variables, objective and constraints are scaled automatically.
REMAXO 40   Maximum number of iterations allowed at the optimizer level.
CONSECO 3   The number of consecutive iterations for which the absolute and/or relative convergence criteria must be met to indicate convergence at the optimizer level.
CONVERGO 1 1 Satisfy relative or absolute convergence
2 Satisfy relative and absolute convergence
CONSECS 2   The number of consecutive iterations for which the absolute and/or relative convergence criteria must be met to indicate convergence at the strategy level.
CONVERGS 1 1 Satisfy relative or absolute convergence
2 Satisfy relative and absolute convergence
REMAXS 20   Maximum number of iterations allowed at the strategy level.
SVSEARCH 1 The one-dimensional search method to be used in the Sequential Quadratic Programming method at the strategy level.
1 Golden-section search method
2 Golden-section search+polynomial interpolation
3 Bounded polynomial interpolation
4 Unbounded polynomial interpolation
ABCONV 0.0001   Absolute convergence criteria for the one-dimensional search when using the Golden Section method.
ABGRESO 0.0001   Maximum absolute fractional change in the objective between two consecutive iterations to indicate convergence in optimization.
ABCONO 0.0001   Absolute fractional convergence criterion for the optimization sub-problem when using sequential minimization techniques.
ABGRESS 0.0001   Same as ABGRESO but used at the strategy level.
RECONV 0.005   Relative convergence criteria for the one-dimensional search when using the Golden Section method.
PROGRESO 0.001   Maximum relative change in the objective between two consecutive iterations to indicate convergence in optimization.
RECONO 0.01   Relative convergence criterion for the optimization sub-problem when using sequential minimization techniques.
PROGRESS 0.001   Same as PROGRESO, but used at the strategy level.
REDELOI 0.1   Relative change in the objective function attempted on the first optimization iteration. Used to estimate initial move in the one-dimensional search. Updated as the optimization progresses.
ABDELOI 1000   Absolute change in the objective function attempted on the first optimization iteration. Used to estimate initial move in the one-dimensional search. Updated as the optimization progresses.
REDELXI 0.01   Maximum relative change in a design variable attempted on the first optimization iteration. Used to estimate the initial move in the one-dimensional search. Updated as the optimization progresses.
ABDELXI 0.2   Maximum absolute change in a design variable attempted on the first optimization iteration. Used to estimate the initial move in the one-dimensional search. Updated as the optimization progresses.
EXTRAP 5   Maximum multiplier of the search step in the one-dimensional search using polynomial interpolation/extrapolation.
SCALO 1.0   The user-supplied value of the scale factor for the objective function if the default or calculated value is to be overridden.
SCALMIN 0.001   Minimum numerical value of any scale factor allowed.
CTOL -0.03   Constraint tolerance in the Method of Feasible Directions or the Modified Method of Feasible Directions. A constraint is active if its numerical value is more positive than CTOL
CLTOL -0.005   Same as CTOL but for linear constraints.
CLTOLMIN 0.001   Same as CTOLMIN, but for linear constraints.
CTOLMIN 0.01   Minimum constraint tolerance for nonlinear constraints. If a constraint is more positive than CTOLMIN, it is considered to be violated.
RIMOVE 0.2   Initial relative move limit. Used to set the move limits in sequential linear programming, method of inscribed hyperspheres and sequential quadratic programming as a fraction of the value of the unknowns.
SQPIP 10.0   Initial penalty parameter in Sequential Quadratic programming.
FRACMOV 0.95   Move fraction to avoid constraint violations in Sequential Quadratic Programming.
STOL 0.001   Tolerance on the components of the calculated search direction to indicate that the Kuhn-Tucker conditions are satisfied.
PUSHOFF 0.1   Nominal value of the push-off factor in the Method of Feasible Directions.
VMULT 2.618034   Multiplier of the search step in the one-dimensional search to find bounds on the solution.
ZRO 0.00001   Numerical estimate of zero on the computer. Usually the default value is adequate. If a computer with a short word length is used, ZRO=10E-4 may be preferred.
